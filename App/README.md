# App

Entry point del proyecto Xcode.

Cuando crees el proyecto en Xcode:
- Product Name: `HackNacional2026` (o el que decidan)
- Interface: **SwiftUI**
- Language: **Swift**
- Storage: **SwiftData** (si lo usan)
- Bundle ID sugerido: `mx.up.ioslab.hacknacional2026`
- Deployment Target: **iOS 18.1** (para Foundation Models)

Mueve los archivos generados (`HackNacional2026App.swift`, `ContentView.swift`) aquí y mantén la carpeta del Xcode `.xcodeproj` en la raíz del repo.
