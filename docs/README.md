# docs

Material de presentación.

- `Pitch.key` — Keynote del comité organizador (llenar antes del cierre)
- `screenshots/` — capturas para el pitch
- `demo-script.md` — guion del demo (10 min + 5 Q&A)
- `team.md` — datos del equipo, asesor, universidad

Plantilla del guion (rellenar):
- 0:00–1:00 Gancho + problema
- 1:00–2:30 Usuario / persona
- 2:30–4:00 Solución y por qué HCAI
- 4:00–7:00 **Demo en simulador o iPhone**
- 7:00–8:30 Stack técnico (frameworks + on-device AI)
- 8:30–10:00 Impacto, roadmap, cierre
